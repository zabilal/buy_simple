# Buy Simple 🛍️

A modern, clean, and intuitive e-commerce mobile application built with Flutter. Buy Simple offers a seamless shopping experience with a focus on simplicity and performance.

## ✨ Features

- 🏠 Clean and intuitive user interface
- 🔍 Product browsing and search
- 🛒 Shopping cart functionality
- 💳 Secure checkout process
- 📱 Responsive design for all screen sizes
- ⚡ Fast and smooth performance

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (latest stable version)
- Android Studio / Xcode (for emulator/simulator)
- VS Code or Android Studio (recommended IDEs)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/zabilal/buy_simple.git
   cd buy_simple
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```
